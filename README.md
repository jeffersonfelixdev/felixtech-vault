# Felixtech Vault

## Getting Started

### Development

To run development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Production

To build:

```bash
npm run build
```

To start:

```bash
npm start
```
